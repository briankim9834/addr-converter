# 부가세 계산기 + 네비게이션 개편 설계 스펙

**날짜:** 2026-04-07
**목표 키워드:** 부가세계산기, 부가세 계산, 공급가액 계산
**URL:** `doguham.kr/vat`

---

## 1. 개요

도구함의 두 번째 서비스로 부가세 계산기를 추가한다. 동시에 헤더 네비게이션을 햄버거+사이드바 방식으로 개편하고, 각 도구 페이지에 Pill 탭 (계산기 | 가이드 | FAQ)을 도입한다.

### 3-2. 사이드바 드로어 (HamburgerMenu.tsx — 신규)

- 우측에서 슬라이드인 (오버레이 포함)
- 햄버거 버튼 클릭 시 열림 / 오버레이 클릭 또는 X 클릭 시 닫힘
- `'use client'` 컴포넌트

**드로어 구성:**
```
┌─────────────────┐
│ 도구 선택        │
├─────────────────┤

│ 📅 D-day 계산기  │ ← 준비중 (비활성)
```

- 현재 도구는 `사용중` badge + 파란색 강조
- 준비중 도구는 opacity 낮춤, 클릭 불가

## 4. 도구 공통 UX — Pill 탭

모든 도구 페이지 상단에 Pill 탭을 배치한다. 탭 클릭 시 URL이 변경된다.

```
[ 🧾 계산기 ]  [ 📖 가이드 ]  [ ❓ FAQ ]
```

- 활성 탭: `bg-blue-500 text-white`
- 비활성 탭: `bg-slate-100 text-slate-600`
- 탭 전환 = Next.js Link로 구현 (URL 변경 → SEO 크롤 가능)

**공통 컴포넌트:** `components/ToolTabs.tsx`

```tsx
interface ToolTabsProps {
  tabs: { label: string; href: string }[]
  currentPath: string
}
```

영문주소 변환기(`/address`)에도 동일하게 적용한다.

## 5. 부가세 계산기

### 5-1. 계산 로직 (순수 클라이언트, API 없음)

| 모드 | 입력 | 계산 |
|------|------|------|
| 공급가액 기준 | 공급가액 | 부가세 = 공급가액 × 0.1 / 합계 = 공급가액 × 1.1 |
| 합계금액 기준 | 합계금액 | 공급가액 = 합계 ÷ 1.1 / 부가세 = 합계 - 공급가액 |

- 원 단위 반올림 (`Math.round`)
- 숫자 입력 즉시 실시간 계산 (버튼 없음)
- 숫자 입력 시 천 단위 콤마 자동 포맷

### 5-2. UI 구성

```
[ 🧾 계산기 ]  [ 📖 가이드 ]  [ ❓ FAQ ]

[ 공급가액 → 부가세 ]  [ 합계금액 → 부가세 ]   ← 계산 모드 탭

공급가액 입력
┌─────────────────────────────┐
│ 1,000,000                원 │  ← 실시간 입력
└─────────────────────────────┘

┌─────────────────────────────────┐
│ 공급가액   1,000,000 원   [복사] │
│ 부가세      100,000 원   [복사] │
│ 합계금액  1,100,000 원   [복사] │  ← 파란색 강조
└─────────────────────────────────┘

[ 초기화 ]
```

- 복사 버튼: 숫자만 복사 (콤마·'원' 제외)
- 복사 후 버튼 텍스트 "복사됨" 표시 (1.5초)
- 입력값 없으면 결과란 `—` 표시

### 5-3. 파일 구성

```
app/vat/
  page.tsx              # 서버 컴포넌트 wrapper (메타데이터 + ToolTabs + VatPageClient)
  layout.tsx            # SEO 메타데이터
  guide/
    page.tsx            # 가이드 서버 컴포넌트
  faq/
    page.tsx            # FAQ 서버 컴포넌트 + FAQPage JSON-LD

components/vat/
  VatPageClient.tsx     # 'use client' — 실시간 계산 UI 전체
```

## 6. SEO

### 6-1. 메타데이터

```ts
// app/vat/layout.tsx
title: '부가세 계산기 — 공급가액·합계금액 자동 계산 | 도구함'
description: '공급가액 또는 합계금액을 입력하면 부가세(10%)를 자동으로 계산해드립니다. 세금계산서 작성, 견적서 정리에 바로 활용하세요.'
canonical: 'https://doguham.kr/vat'
keywords: ['부가세계산기', '부가세 계산', '공급가액 계산', '합계금액 부가세', '세금계산서']
```

### 6-2. 가이드 페이지 (`/vat/guide`)

섹션 구성:
1. 부가세란? (개념)
2. 공급가액 vs 합계금액 차이
3. 3단계 사용법
4. 활용 예시 (세금계산서, 견적서, 프리랜서 계산)
5. CTA → `/vat` 링크

### 6-3. FAQ 페이지 (`/vat/faq`)

FAQPage JSON-LD 포함. 예상 FAQ 항목:
1. 부가세율은 항상 10%인가요?
2. 공급가액과 합계금액의 차이는?
3. 간이과세자도 이 계산기를 쓸 수 있나요?
4. 역산(합계금액 기준) 계산은 어떻게 하나요?
5. 결과를 어떻게 복사하나요?
6. 소수점은 어떻게 처리하나요?
7. 부가세 면세 품목은?
8. 계산 결과가 1원 차이 나는 이유는?
9. 세금계산서 작성 시 어떤 값을 입력하나요?
10. 이 계산기는 무료인가요?

## 8. 비기능 요구사항

- 외부 API 없음 — 순수 클라이언트 계산

- 기존 `/address/guide`, `/address/faq` URL 유지 (SEO 보호)
- 모바일 우선 반응형 (max-w-lg 기준)
